# Heliostat-Placement-for-Solar-Tower
MATLAB implementation of the heliostat field design and optical performance from the paper "Design Calculations of Heliostat Field Layout for Solar Thermal Power Generation" (Hnin Wah &amp; Nang Saw Yuzana Kyaing, Mandalay Technological University, Myanmar). Built for a renewable energy/CSP coursework assignment to replicate the paper's design.
